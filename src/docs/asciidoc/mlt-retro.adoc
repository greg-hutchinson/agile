## Warmup Exercise
[%step]
- Your task is to identify the rule to the next number in a list of numbers that I will give you.
- You can ask me only one question - "Is n the next number in this list".
- However, you can ask this **any number** of times.
- When you think that you know the rule you can guess - but you only get 1 guess
- If you are right - you win.
- Here is the start of the list:
    - 2, 4, 6, 8, ...

## Retrospective
[.smaller-bullet]
* Everyone will write down some ideas as stickys related to the topics on the Miro Board
* We will talk about each one quickly to understand what the sticky is about
* We will group them if they are similar
* We will vote on which themes we think are the most important 
* We will break into 2 groups to discuss the top 2 themes. Each group will try and come up with an experiment to try over the next month in which they believe the process can be improved.
