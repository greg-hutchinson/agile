## ROI Calculation

ROI = BusinessValue / Cost

For the Mathematicians in the crowd +
As Cost goes down, ROI goes up + 
(As Cost → 0 then ROI → ∞) 

And conversely +
As Cost goes up, ROI goes down +
(As Cost → ∞ then ROI → 0)
